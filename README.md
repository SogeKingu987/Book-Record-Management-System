# Book Record Management System (C++)

A console-based C++ mini project built as part of a larger learning roadmap towards developing a Mini Library Management System.

## Features
- Add book records
- View all records
- Remove books by ID
- Confirmation handling using enums
- Dynamic storage using STL vectors

## Concepts Practiced
- Multi-file project structure (`.cpp` / `.hpp`)
- Structs and classes
- STL vectors
- Iterators
- Enums
- Function modularization
- Iterator-safe deletion using `erase()`
- Basic program architecture and flow control

## Technologies Used
- C++
- STL Vector
- Iterators
- Enums
- g++
